| Class A | 10.0.0.0 ~ 10.255.255.255     | 10.0.0.0/8     | 255.0.0.0     | 2^24 (약 1,677만) 개 |

| Class C | 192.168.0.0 ~ 192.168.255.255 | 192.168.0.0/16 | 255.255.0.0   | 2^16 (약 6.5만) 개   |
